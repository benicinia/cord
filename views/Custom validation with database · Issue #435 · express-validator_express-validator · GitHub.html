gustavohenke commented on 25 Sep 2017

Eae @barrosfilipe ,

The example you gave as not working is because you're using an arrow function without a return.
value => value is the same as value => { return value }, but value => { value } is simply a function that evaluates value, without returning anything.

In the full example for DB validation, you are trying to return from within an async callback.
In this case, you should use promises instead.
For a detailed explanation, see this popular StackOverflow question.
